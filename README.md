## Cài đặt
1. Clone repo:
   git clone https://github.com/DuyVu1204/Group_10.git
2. Tạo môi trường ảo và cài đặt:
   - Với venv:

     - python -m venv venv

     - Kích hoạt môi trường ảo (Windows): `source ./venv/Scripts/activate`

     - Cài đặt thư viện cho dự án: `pip install -r requirements.txt`
